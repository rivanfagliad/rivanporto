# Color Palette

## Primary Colors

| Role | Color | Hex | Usage |
|------|------|------|------|
| Primary | Charcoal Black | `#111111` | Text, Border, Icons |
| Background | Soft White | `#FAFAF7` | Main Background |
| Surface | Concrete Gray | `#EAEAEA` | Secondary Background, Section |
| Accent | Electric Purple | `#7B5CFF` | Hero, Featured Project, Highlight |
| CTA | Acid Lime | `#D9FF3F` | Button, Badge, Active Navigation |

### Optional Status Colors

| Role | Color | Hex |
|------|------|------|
| Success | Emerald Green | `#29CC74` |
| Warning | Orange | `#FF8A00` |
| Error | Coral Red | `#FF5A5F` |
| Info | Sky Blue | `#3BE8FF` |

# Shadow

Neo Brutalism menggunakan shadow berbentuk offset (tanpa blur).

```css
box-shadow: 8px 8px 0px #111111;
```

Hover

```css
transform: translate(-4px, -4px);

box-shadow: 12px 12px 0px #111111;
```

---

# Border Radius

```css

```

# Animation

Neo Brutalism menggunakan animasi sederhana.

Hover

```css
transform: translate(-4px,-4px);
transition: all .2s ease;
```

Scroll

- Fade In
- Slide Up

Tidak menggunakan parallax atau animasi yang berlebihan.
